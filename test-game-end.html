🎮 Rocket Wars - End Game Test

This is a quick test page to verify the end game functionality works correctly.

Main features to test:

  • ✅ Game ends when only one player remains
  • ✅ Game ends when max rounds are completed
  • ✅ Results panel appears with ranked players
  • ✅ Camera focuses on winner
  • ✅ Restart works with R key or clicking text
  • ✅ Results panel is locked from scrolling
  • ✅ Landscape can still be scrolled behind panel

🧪 Testing Steps:

  1. Open index.html in your browser
  2. Set up a quick game:
    • Number of Players: 2
    • Turn Time: 10 seconds (for quick testing)
    • Rounds: 1 (to test max rounds completion)
  3. Play the game and either:
    • Option A: Kill the other player to test "last player standing" end condition
    • Option B: Let the single round complete without kills to test "max rounds" end condition
  4. Verify the Results panel appears with:
    • Player rankings (alive > killed, then by health)
    • Health percentages
    • Winner highlighted in gold
    • Status icons (🏆 for alive, 💀 for dead)
    • Restart instruction
  5. Test restart functionality:
    • Press R key → should reload page
    • Click "Press R to restart game" text → should reload page
  6. Test camera and scrolling:
    • Camera should focus on winner's turret
    • Results panel should stay fixed on screen
    • Dragging should still scroll the landscape behind the panel
